How To Avoid Selecting a Branch After Your New Project Has Been Created
  • For github.com URLS:
    • if you put in /tree/{branch_name} after the repo URL, then you won't be asked to select a branch after your new project has been cloned to your computer. The reason why this method works is because it allows you to use not only the available branches in the repo URL, but also the repo's commit branch (and release tag) as well. This can be helpful for if you want to create a project using an earlier version of the repo URL provided.
